#### EPICS Wabash Center
# Rube Goldberg Machine
#### In cooperation with the Art Museum of Greater Lafayette's ArtSmart Program
#### Sponsored by Purdue University, EPICS at Purdue, and Catepillar

***

Repository built to store and manage CAD files for the ArtSmart Rube Goldberg project.
-> ***.STEP*** is the recommended/required format for all design files.

***

**Currently in-progress systems follow the order provided** (alongside directory names)
1. Train Whistle -> train-whistle
2. Train Loop -> train-loop
3. Purdue Sign -> purdue-sign
4. Ball Ramp -> ball-ramp
5. Plane Ride -> plane-ride
6. Ball Return -> ball-return
7. Race Car Track -> race-car-track
